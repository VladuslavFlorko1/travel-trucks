# 🚐 TravelTrucks

TravelTrucks — це вебзастосунок для оренди кемперів, створений з використанням **Next.js** та **TypeScript**.

---

## 🔗 Live Demo

```
https://your-project.vercel.app
```

---

## 📦 Функціонал

* 🔍 Перегляд каталогу кемперів
* 🎯 Фільтрація за:

  * локацією
  * типом кузова
  * двигуном
  * трансмісією
* ➕ Пагінація (Load More)
* 📄 Сторінка деталей кемпера
* 🖼 Галерея зображень
* ⭐ Відгуки користувачів
* 📬 Форма бронювання
* 🔔 Нотифікації після відправки форми

---

## 🛠 Технології

* **Next.js (App Router)**
* **TypeScript**
* **TanStack Query**
* **Axios**
* **React Hooks**
* **React Hot Toast**
